DeepResearch 多智能体行业调研系统

面向 ToB 行研场景的 AI-Native 生产力工具 —— 从用户提问到万字调研报告，端到端自动生成。

技术栈：Python · FastAPI · LangGraph · PostgreSQL · Milvus · Redis · asyncio · SSE · DeepSeek/Qwen · React · TypeScript · Ant Design

个人职责：

  1. 设计 Multi-Agent 状态编排架构 — LangGraph StateGraph + conditional_edge 条件路由
  2. 实现步骤级 SSE 实时推送机制 — asyncio.Queue + 16 种事件类型全覆盖
  3. 搭建 Checkpoint 断点恢复机制 — PostgreSQL JSONB + session_id 跨会话恢复
  4. 构建双模式检索与信源治理链路 — Bocha + Milvus + 可信度评分 + 两层递归溯源
